## 🧪 Tests:

To run tests, just type npm run test.

```bash
  npm run test
```

## 🐋 Container

Just docker-compose up, to up container project. Case you want run local, just npm run dev.

```bash
  docker-compose up
```
```bash
  npm run dev
```
